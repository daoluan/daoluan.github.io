# 搬家啦


2012年5月12日有了自己的独立博客，在博客园的那一段时间里，一直压力有点肿。园里高手如云，以至于写点东西也要小心翼翼的。之前也是一直写技术的东西，很少涉及生活。所以决定给自己换一个比较轻松的环境，写技术，写心情，写生活，记录点点滴滴。
